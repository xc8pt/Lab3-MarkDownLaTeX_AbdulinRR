Markdown в комментариях к коду *
**Требования к программе:**
*   1. Имя файла: FormatDemo.cs
    2. Логика:- Запрашивает у пользователя два числа;- Выполняет их сложение;- Выводит результаты в форматированном виде.*

```csharp
Console.Write("Введите первое число: ");
int num1 = int.Parse(Console.ReadLine());
Console.Write("Введите второе число: ");
int num2 = int.Parse(Console.ReadLine());
int sum = num1 + num2;
Console.WriteLine($"**Результат операций: {sum}**");
```
