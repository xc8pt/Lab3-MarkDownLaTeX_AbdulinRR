### Списки
#### Маркированный список:
- первый пункт
- второй пункт
- третий пункт
#### Нумерованный список:
1. первый пункт
2. второй пункт
3. третий пункт
#### Вложенный список:
- пункт 1
  - подпункт 1.1
  - подпункт 1.2
